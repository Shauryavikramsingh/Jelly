# 🍮 Jelly: The Lost Adventure
**A game project by Shaurya Vikram Singh & Veer Vikram Singh**

## 📖 The Story
Jelly is a physics-based platformer about a tiny, wobbly hero who got separated from the pack in a giant, dangerous kitchen. 

## 🎮 Game Info
* **Status:** In Development 🚀
* **Main Dev:** Shaurya Vikram Singh (Logic & Physics)
* **Level Design:** Veer Vikram Singh (Obstacles & Traps)
* **Goal:** Navigate through "The Countertop" and "The Sink" levels to find home!

## 🛠️ Tech Stack
* Built on **Scratch** (or your platform)
* Focused on **Squishy Physics** and fast-paced jumping.
